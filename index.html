Extend Minecraft's visual elements.

There are two Complementary shaderpacks that are both complex visual overhauls for Minecraft Java Edition.
Complementary Version 4 originally released in 2020 as a BSL edit. Complementary Reimagined is a new shaderpack that was released two years later.
The main difference between them is the stylistic direction. Reimagined doesn't discard Minecraft's visual design decisions while Version 4 does.
But there are still many little differences between the two packs, so we made this table that cover the most important aspects.
