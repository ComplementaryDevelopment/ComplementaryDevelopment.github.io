Mod Issues

Mod Issues With Shaderpacks

Shaderpacks rework the visuals of Minecraft. This means incompatibilities with mods that do complex fancy visual effects can be unavoidable.
But luckily, we know ways to fix a lot of the common problems.

Known Fixes

- If you are using "Complementary Shaders v4": enable "Compatibility Mode" in the main menu of Shader Options.
- If you are using "Complementary Reimagined": set "RP Support" to "Basic" in the main menu of Shader Options.
- Ensure that you are using the newest Optifine or Iris version for your Minecraft version.
- If you are using the "Oculus" mod: please consider switching to Optifine as Oculus is unsupported.
- If you have the "Astral Sorcery" mod for Mc 1.12: Open the mod config file, set "S:weakSkyRenders" to "0". (Like This)
- If you have the "Astral Sorcery" mod for Mc 1.15: Open the mod config file, set "weakSkyRenders" to "[0]". (Like This)
- If you have the "Astral Sorcery" mod for Mc 1.16+: Open the mod config file, set "skyRenderingEnabled" to "[]". (Like This)
- If you have the "Advanced Rocketry" mod: Open the mod config file, set "B:overworldSkyOverride" to "false".
- If you have the "Botania" mod: Open the mod config file, set "B:fancySkybox.enable" AND "B:shaders.enabled" to "false".
- If you have the "Computer Craft: Tweaked" mod: Open the mod config file, set "S:monitor_renderer" to "gui.computercraft:config.peripheral.monitor_renderer.vbo".
- If you have the "Thermal Dynamics" mod: Add this mod to your modpack.
- If you have the "The Midnight" mod for Mc 1.12: Open the mod config file, set "B:rift_shaders" to "false".
- If you have the "The Betweenlands" mod: Open the mod config file, set "B:use_shader" to "false".
- If you have the "Rats" mod: Go to Video Settings > Performance, and set "Fast Render" to OFF.
- If you have the "CoFH Core" mod: Open the mod config file, set "Render Area Block Breaking" to "false".
- If you have the "Flywheel" mod: type "/flywheel backend off" in the in-game chat
- If you have the "Snow Real Magic" mod: Make sure to use SnowRealMagic version 1.16.3-2.2.2 or above.
- If you have the "BetterEnd Forge Port" mod: Update your mod to the latest version.
- If you have the "Entity Culling" mod: Open the mod config file, set "entityShadowsCullingLessAggressiveMode" to "true".
- If you have the "Mana and Artifice" mod: Open the mod config file, set "hudMode" to "1".
- If you have the "Psi" mod: Open the mod config file, set "useShaders" to "false".
- If you have the "Extra Utilities" mod: Open the mod config file, set "I:"Deep Dark"" to "10".
- If you have the "Clientcommands" mod for Mc 1.16 or below: Only known fix is removing the mod.
- If you have the "The Aether" mod for Mc 1.16: Open the mod config file, set "Disables Aether custom skybox" to "true".

Still Having Issues?

If you are still having issues after doing all of the above, you have more options:
- This seems dumb, but it is very common to miss a couple of fixes in the list. We recommend you to go over the list again if you are still having issues.
- If you are having a hard time, you can join the Complementary Discord Server to get help from our welcoming community.
